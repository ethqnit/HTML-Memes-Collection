# HTML Meme Gallery – Pure HTML + CSS Fun!

A super simple, beginner-friendly meme collection page built with **only HTML and CSS** – no JavaScript, no frameworks, just good old web fundamentals!

Perfect as a learning project or just for laughs.

## Features

- Fully responsive (works on phone/tablet/desktop)
- Animated GIF background (yes, the whole page moves!)
- Clean navigation menu with hover effects
- 10 classic and funny memes/GIFs
- 100% vanilla – no JavaScript, no libraries
- Simple, readable code – perfect for beginners

<br>
